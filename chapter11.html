AI Jargon Demystified (Glossary)

Understanding the terminology of modern AI

Architecture Terms

Transformer

Neural architecture using self-attention for sequence processing

Attention Head

Parallel attention computations in transformer layers

Feed-Forward Network

Dense layers within transformer blocks

Embedding

Dense vector representations of discrete tokens

Training Concepts

Pre-training

Initial training on large, unlabeled text corpora

Fine-tuning

Adaptation to specific tasks with labeled data

RLHF

Reinforcement Learning from Human Feedback for alignment

Gradient Descent

Optimization algorithm for parameter updates
